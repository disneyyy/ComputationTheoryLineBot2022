### 七海麻美
生日：11月13日  
星座：天蠍座  
聲優：悠木碧  
七海麻美是由宮島禮吏創作的漫畫《出租女友》及其衍生作品的登場角色。  
練馬大學一年級生，曾與木之下和也交往一個月（實際上是隨便答應的），隨即爽快地甩了他。表面上性格爽朗，實際上極度腹黑且善妒。  
是個家教嚴格的大小姊。因為小時候最心愛的娃娃被父親丟棄，以及被迫與喜歡的對象分手，造就了麻美腹黑的性格，願景是拆散世界上所有的情侶。
